O que é CSS?

Definição – CSS é a abreviação em inglês do termo Cascading Style Sheet, ou, em português, folhas de estilo em cascata.

“Folha de estilo em cascata é um mecanismo simples para adicionar estilos (fontes, cores, espaçamentos, etc...) a um documento HTML.”

O HTML deve estruturar os objetos e o CSS deve apresentá-los na página;

O que é HTML?

Linguagem de marcação de textos. HTML significa Hypertext Markup Language (linguagem de marcação de hipertexto). Composta por elementos chamados tags ou rótulos ou marcadores. Estes marcadores definem a forma como o navegador apresenta a informação na página web. O navegador é o software mais importante para a exibição e renderização de um documento HTML

HTML não é linguagem de programação.

HTML5 é muito flexível em relação ao uso de elementos e à escrita do documento HTML, o que pode dificultar o entendimento e a legibilidade do código criado.
